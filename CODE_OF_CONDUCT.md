# Code of Conduct

## Our Standard

This project aims to be a respectful, collaborative, and welcoming space.

Participants are expected to:

- communicate respectfully and professionally
- give constructive feedback
- assume good intent while still being clear and honest
- respect differing viewpoints and experience levels
- focus on what is best for the project and its community

Unacceptable behavior includes:

- harassment, discrimination, or personal attacks
- insulting, demeaning, or hostile language
- deliberate disruption of discussion or review processes
- publishing private information without permission

## Scope

This code of conduct applies to project spaces, including issues, pull requests, code review, and other project-related discussions.

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that do not align with this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, report it to the project maintainers through the repository’s normal communication channels.
